# ✅ Verification Checklist

## Before Running

- [ ] MongoDB is running and accessible
- [ ] `.env` file has correct `MONGO_URI` and `DB_NAME`
- [ ] Backend dependencies installed: `npm install`
- [ ] Frontend dependencies installed: `cd frontend && npm install`

## Backend Verification

Run: `npm start` or `npm run dev`

- [ ] Server logs: "Server running on port 5000"
- [ ] Server logs: "DB Connected"
- [ ] Health check works: `http://localhost:5000/` returns `{"success": true, "message": "Server is running"}`
- [ ] CORS headers are enabled

## Frontend Verification

Run: `npm run client`

- [ ] Vite dev server starts on port 3000
- [ ] Browser automatically opens or manually go to `http://localhost:3000`
- [ ] No console errors about CORS
- [ ] Page title shows "Ecommerce"

## UI Verification

- [ ] Header displays "User Management" title

- [ ] Form section visible with fields: Name, Email, Phone, Address

## If Something Fails

### Check Backend Logs
```bash
# Look for error messages when starting
npm start
```

### Check Frontend Console
Press F12 in browser → Console tab → Look for red error messages

### Check Network Requests
Press F12 → Network tab → Try creating a user → Look for failed requests

### Verify Connectivity
```bash
# Test if backend is running
curl http://localhost:5000/

# Should return:
# {"success":true,"message":"Server is running"}
```

### Reset Everything
```bash
# Stop both services (Ctrl+C)
# Kill any lingering processes
netstat -ano | findstr :3000  # Find process
taskkill /PID <PID> /F         # Kill it

# Same for port 5000
netstat -ano | findstr :5000
taskkill /PID <PID> /F

# Clear browser cache
# Then start fresh
npm run dev
npm run client
```
